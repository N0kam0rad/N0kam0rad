## Status vocabulary

```text
Done        = usable, shipped, or complete enough to demo honestly
In progress = actively building, beta, prototype, testing, or drafting
Planned     = scoped or committed, but not started yet
Idea        = early concept only
```

---

## My build lane

```text
idea spark      → product framing
voice notes     → specs, prompts, workflows
AI assist       → prototype, content, demo, iteration
human taste     → story, structure, usability, polish
```

I’m especially interested in:

- AI-assisted product prototyping
- Presales storytelling and demo strategy
- Workflow automation
- Product specs that make fuzzy ideas real
- Creative systems with repeatable style and canon
- Turning personal pain points into useful tools
